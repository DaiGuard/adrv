# adrv_gazebo

ADRV(Auto Drvie Robot Vehicle) spawn robot and world in gazebo.

----------------------------------------------------------------

## Launch
----------------------------------------------------------------

- ### ```adrv_course.launch```

  spawn adrv robot description and course world in gazebo.

  depends:
  
  - ```spawn_ardv.launch```
  - ```course.launch```

- ### ```spawn_ardv.launch```

  spawn adrv robot description in gazebo.

- ### ```course.launch```

  spawn course world in gazebo.
